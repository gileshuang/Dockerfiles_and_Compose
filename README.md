## Podman-Compose

一些我自己在用的 podman-compose 文件，语法和 docker-compose 基本一样，但有些细微差别。

> 这些 podman-compose 文件只能保证在我自用的版本的 podman 中可用，docker 环境中可能需要进行修改。
> 有些 docker-compose 中的语法，在 podman 中也不可用，比如 docker-compose 中支持在 volume 映射中使用变量，但 podman 中实测有点问题，所以这几个 podman-compose 文件就显得很不优雅。

其中可能会包含一些我自己的配置，特别是 volume 映射路径。如有需要取用，请自行修改。
